# 模块目录说明

当前阶段已完成：

- `health`：系统健康检查
- `auth`：管理员登录、登出、当前登录态查询
- `user`：管理员初始化、当前账户密码修改
- `post`：管理员侧文章创建、列表、详情、更新基础接口
- `category`：管理员侧分类创建、列表、更新接口，支持基础排序字段
- `tag`：管理员侧标签创建、列表、更新接口
- `public-post`：公开侧文章列表、详情接口，支持基础分页与关键词搜索

后续计划模块：

- `post`：文章 CRUD 与发布流转补全
- `category`：后续可补充删除、详情、批量排序能力
- `tag`：后续可补充删除、详情、关联文章统计增强
- `comment`：评论提交与审核
- `setting`：站点设置
- `search`：搜索能力
