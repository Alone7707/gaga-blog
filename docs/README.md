# 开源博客产品文档

本目录用于沉淀该开源博客产品的产品、技术、协作与模块规划文档。

## 文档索引

- `PRD-开源博客产品.md`：产品需求文档
- `技术选型与架构.md`：技术方案、架构边界、核心数据模型
- `模块拆分与协作规范.md`：模块拆分、开发顺序、Git 提交规范

## 当前产品目标

打造一个面向个人开发者、小团队与内容创作者的开源博客系统，具备：

- 内容发布与管理
- 标签与分类
- 评论与审核
- 搜索与 SEO
- 后台管理
- 可扩展主题能力
- 轻量部署与低维护成本

## 当前技术方向

- 前端：Vue 3 + Vite + Vue Router + Pinia
- 后端：Node.js + NestJS（如实现阶段评估过重，可降级为 Fastify）
- 数据库：SQLite + Prisma
- 鉴权：JWT + HttpOnly Cookie

## 协作约束

- 每完成一个独立模块，必须提交一次 Git commit
- 文档与代码都应先落仓库，再进入下一个模块
- 不在文档、代码、git config 中持久化 PAT 等敏感凭据
